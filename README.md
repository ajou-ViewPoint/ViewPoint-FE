# ViewPoint - Frontend

대한민국 국회 정보 시각화 프로젝트의 **프론트엔드 레포지토리**입니다.  
Organizations 레벨의 전체 프로젝트 소개는 [여기]에서 확인할 수 있습니다.

---

## 📋 이 레포지토리의 역할

국회 의정 데이터를 사용자가 쉽게 이해하고 상호작용할 수 있도록 UI를 구현할 예정입니다.  
React를 기반으로 사용자 경험(UX)에 초점을 맞춰 개발합니다.

---

## ✨ 주요 구현 화면 (Features)

-   📊 **랜딩 페이지 (대시보드)**  
    국회의원 이념 공간, 최신 법안 현황 등 핵심 데이터를 시각화

-   ⚖️ **법률안 분석**  
    법안의 상세 정보 및 표결 결과를 차트와 지도로 시각화

-   🏢 **위원회 및 정당**  
    위원회 구성, 정당별 이념 분포 등을 시각적으로 표현

-   👥 **국회의원 정보**  
    국회의원 개인의 상세 정보 및 의정 활동 데이터를 제공

---

## 🛠️ 기술 스택 (Tech Stack)

-   **Frontend Framework**: React (Vite 기반)
-   **상태 관리**: Jotai(예정)
-   **빌드 & 배포**: AWS(예정)
